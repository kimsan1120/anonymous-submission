아래 내용을 바탕으로 한국어 콜센터 전화 통화 ASR STT 느낌의 두 사람 대화로 바꿔줘.

조건:
- 고객과 상담원이 통화하는 느낌으로 만들 것
- 실제 콜센터 통화처럼 질문하고 답하고 다시 묻는 흐름으로 만들 것
- 설명문이나 지문 없이 실제 발화만 남길 것
- 문장은 짧고 단순하게 만들 것
- 전체 텍스트 길이는 크게 줄이지 말 것
- 너무 자연스러운 일상 수다처럼 만들지 말고 상담형 흐름으로 만들 것
- 원문의 주제와 핵심 내용만 대체로 유지하면 됨
- 세부 표현은 콜센터 통화처럼 자연스럽게 바꿔도 됨
- 과한 감정 표현, 과한 공감, 과한 리액션은 넣지 말 것
- 출력은 하나의 연속된 텍스트로만 작성할 것
